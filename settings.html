Azure AD Configuration

To enable the "Record Runs" feature, you need to provide an Azure AD Application Client ID.

Steps:

  1. Go to Azure Portal
  2. Navigate to Azure Active Directory → App registrations
  3. Copy your Application (client) ID
  4. Paste it below
This is the Application (client) ID from your Azure AD app registration
Leave as "common" for multi-tenant, or enter your specific tenant ID
